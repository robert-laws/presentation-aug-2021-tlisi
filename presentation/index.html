Maximizing Technology for a Course on Data Visualization

New and Revised Pedagogical Tools • TLISI 2021

Robert Laws
Technology & Instruction Librarian

1. Course Details

1. Course Details

  • INAF 092 - Introduction to Data Visualization
    (1-Credit Course) - Spring 2021 with 18 students
  • Two Main Course Goals:
    1. Core data literacy concepts, and
    2. How to use Tableau (data visualization software)

2. Challenges

2. Challenges

  • Course was only 6 weeks long - ambitious amount of content
  • Lab setting was unavailable - had to do everything online
  • Balance between theoretical and practical content

3. Pedagogy

3. Pedagogy

Core Pedagogies included:

  • On Demand Learning
  • Multiple Content Delivery Strategies with Technology

On Demand Learning

Includes content or instruction students could access/request outside of class time

  • Asynchronous media content (YouTube videos, data visualization website)
  • Asynchronous consultations (1-on-1 Zoom meetings)
  • 60 second help videos (homework examples, quick help videos)
  • Library & OER resources

Multiple Content Delivery Strategies with Technology

  • Course Texts (E-book versions)
  • YouTube videos
  • Custom website on data visualization
  • Homework solutions (short videos)
  • 1-on-1 consultations via Zoom
  • Online Library Guide (dataset resources)

4. Course Content Examples

4. Course Content Examples

  • Data Viz Cafe - custom website on data visualization
  • YouTube Playlist - 24 videos
  • Library Guide on Datasets & Data Visualization

5. Technologies Used

5. Technologies Used

  • Tableau for Students - Course Software
  • React - Website Development
  • Camtasia - YouTube Videos
  • Snagit - 60 Second Videos

6. Outcomes

6. Outcomes

  • Students made frequent use of asynchronous 1-on-1 consultations via Zoom and short videos
  • Custom website and YouTube videos were used, but not as much as expected
  • Solutions provided were satisfactory to meet each students learning preferences
  • Extensive preparation and time was required to implement multiple technologies
